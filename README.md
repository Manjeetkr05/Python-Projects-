# 🌦️ Weather Data Analyzer (Python)

A Python-based application that fetches real-time weather data using the OpenWeather API, processes it using Pandas, and performs basic data analysis.

---

## 🚀 Features

- Fetch real-time weather data for multiple cities
- Displays:
  - Temperature (°C)
  - Humidity (%)
  - Weather condition
- Stores data in a structured Pandas DataFrame
- Performs analysis:
  - Average temperature
  - Hottest city
- Exports data to a CSV file
- Handles invalid inputs and network errors

---

## 🛠️ Tech Stack

- Python
- Pandas
- Requests (API integration)
